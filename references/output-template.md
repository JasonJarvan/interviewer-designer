# Interviewer Manual Template

Use this structure for a full manual.

## 1. Direction Confirmation

If not already specified, ask the interviewer to confirm:

- Primary hard-skill directions and weights.
- Secondary soft-skill directions and weights.
- Whether public resume/project verification is allowed.
- Whether the interview should include live coding, system design, project deep dive, or open-ended discussion.

## 2. Candidate Claim Map

Summarize:

- Candidate level and likely seniority.
- Claimed projects and technologies.
- Claimed ownership and quantified outcomes.
- High-risk or high-value claims to verify.

## 3. Public Verification Report

For each major project:

- Resume claim.
- Public evidence.
- Support level: strong support, partial support, unverifiable, conflicting.
- What can be verified.
- What cannot be verified.
- Interview verification probes.

## 4. One-Hour Interview Flow

Use minute-level planning. Default shape:

- 0-8 min: boundary confirmation.
- 8-20 min: project architecture and contribution validation.
- 20-40 min: core hard-skill probes.
- 40-50 min: selected design/coding/depth probe.
- 50-57 min: soft-skill probes.
- 57-60 min: open-ended concepts or candidate questions.

Adjust based on the interviewer's weights.

## 5. Boundary Confirmation

Always include questions for:

- Overall project architecture.
- Candidate-owned modules.
- Non-owned modules.
- Organization and reporting structure.
- Decision authority.
- Deployment and operations involvement.
- Incidents or production issues personally handled.

## 6. Question Pool

Split into:

- Candidate-selected baseline questions.
- Interviewer-selected depth questions.
- Bonus questions.
- Risk-verification questions.

For each question include:

- Question.
- Assessment target.
- Applicable ownership boundary.
- Expected answer.
- Acceptable variants.
- Strong signals.
- Risk signals.
- Follow-up chain.
- Scoring rubric.

## 7. Live Evaluation

For each candidate answer:

- Matched points.
- Missing points.
- Contradictions.
- Initial score.
- Likely reason for mismatch.
- Clarifying follow-up.
- Revised score after follow-up.

## 8. Final Assessment

Report:

- Hard-skill score.
- Soft-skill score.
- Authenticity risk.
- Seniority calibration.
- Hire/no-hire recommendation.
- Key evidence behind the recommendation.
